Platform Mapping

When determining the platform of a ROM or image (which is later used when determining the game title), only the "Unique File Extensions" are used. All other extensions are ignored as they will limit the ability of Gaseous to determine the game title (see https://github.com/gaseous-project/gaseous-server/wiki/ROM-Metadata-Matching for more information on how matching works).

This list is pre-populated with some of the more common platforms. New platforms will appear in this list as titles are added.
